# icemacs

I love Emacs!

### Usage

Install

    git clone https://github.com/kenzo44/icemacs.git ~/.emacs.d

Compile (not required, but recommended)

    cd ~/.emacs.d
    make compile

Run

    emacs

    
### Make Commands

**clean**: Delete compiled files

    make clean

**compile**: Byte compile for performance (Recompile required when new changes are made)

    make compile
